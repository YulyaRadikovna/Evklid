Сайт “Евклид”.

Сайт по макету из Figma с применением pixelperfect.
Сайт адаптивный для любых устройств от 320 до 3840 пикселей по ширине.

Количество брейк-поинтов в медиа запросах не превышает семи (7).

В проекте присутствует:
- фавикон
- бургер-меню
- кнопка поиска информации на сайте
- свайпер
- табы (переключение табов)
- секция с вопросами (аккордеон)
- focus, hover, active
- flex
